# Outreach
Discussion about marketing and reaching out to both members inside the community and out.

If you're passionate about telling the world all about Backdrop, engage with others who love the project and wish more people could see how awesome it is!

**Where to start...**

1) If you want to discuss things, head over to the Discussions tab.

2) If you want to add or ammend a policy, create an issue in the Issue tab.

3) If you want to see the progress of any outstanding issues, check out the Project tab.
